## Windows Build

The Windows build of Paxon has been released.
When installed through the installer or package manager,
the executable is automatically added to the system `PATH`,
allowing it to be accessed from Command Prompt, PowerShell, and other supported shells.

> [!WARNING]
> **Compatibility Notes:**
> * **Architecture:** The Windows build supports **x86_64 (Intel/AMD)** architectures. ARM-based Windows devices (e.g., Snapdragon X / Surface Pro X) are not officially supported.
> * **OS Version:** Windows 10 (version 1903 or later) or Windows 11 is recommended. Older versions may encounter runtime compatibility issues.

See [platform differences](/Docs/Linux/platform-differences.md) for more details.
