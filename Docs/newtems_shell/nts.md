# newtem's Shell Script

`.nts` is the script format used by newtem’s shell. 
It is designed to provide a simple and flexible way to
execute commands, configure the environment, and more within the shell.

## Overview

An `.nts` file can contain:
 + Command execution sequences
 + Environment configuration
This allows .nts to function both as a configuration file.

 // For scripting, using NetDust with the Noctura engine is recommended.

## Usage

`key` = `value`
e.g., `ignis_theme = Matcha-Latte`
